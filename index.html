Gittertier Pro Tutorial

Willkommen bei Gittertier Pro!

Diese kurze Einführung zeigt Ihnen, wie Sie die App optimal nutzen können. Folgen Sie den Schritten, um alle Funktionen kennenzulernen.

Produkte scannen

Starten Sie den Scanner, um Barcodes von Produkten zu erfassen. Die Kamera sucht automatisch nach Codes und erkennt Produkte.

Produktinformationen

Hier sehen Sie die Details des gescannten Produkts. Sie können die Menge anpassen und das Produkt zum Warenkorb hinzufügen.

Warenkorb verwalten

Im Warenkorb sehen Sie alle ausgewählten Produkte. Sie können Mengen anpassen oder Artikel entfernen.

Bezahlvorgang

Generieren Sie einen QR-Code, um die Zahlung abzuschließen. Der Kunde kann diesen Code mit seinem Smartphone scannen.

Manuelle Eingabe

Wenn ein Produkt nicht erkannt wird, können Sie es manuell erfassen. Die Daten werden für zukünftige Scans gespeichert.

Einstellungen

Über die Admin-Einstellungen können Sie Öffnungszeiten, Währung und andere App-Einstellungen anpassen.
